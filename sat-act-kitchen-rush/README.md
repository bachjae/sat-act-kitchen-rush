# SAT/ACT Kitchen Rush

A 2D top-down pixel art educational game where students answer SAT/ACT test questions while managing a virtual kitchen.

## Tech Stack

- **Frontend**: React + TypeScript + Vite + PixiJS
- **Backend**: Node.js + Express + Socket.io
- **Database**: Firebase (Auth + Firestore)
- **State**: Zustand
- **Styling**: Tailwind CSS

## Getting Started

### Prerequisites

- Node.js v20+ LTS
- npm v10+

### Development

```powershell
# Start frontend
cd client
npm run dev

# Start backend (separate terminal)
cd server
npm run dev
```

- Frontend: http://localhost:3000
- Backend: http://localhost:5000
